# RemedicaAI---Turning-Waste-into-Medicine-with-Generative-AI
GenAI turns waste (peels, grounds) into drugs &lt;60s. Upload img/txt → fragments → structures → ADMET/EcoScore → RAG validate → PDF. Built: GPT-4o, Llama-3, LangChain, RDKit. Guided: Dr. Hammad Majeed (UMT Lahore). Hackathon 2025. Live: https://remedica-waste-whisperer.lovable.app
